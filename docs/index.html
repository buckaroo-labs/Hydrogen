Hydrogen is a Framework for development and documentation of responsive web applications. It is based on a w3schools example, using w3.css, and has been in sporadic development since 2014.

Hydrogen is designed for small projects that run on inexpensive hosting, e.g. a LAMP stack, and runs on vanilla PHP. Data storage options include SQLite, MySQL, and Oracle.

Hydrogen is designed to support the following high-level functional requirements:

  • Consistent look for multi-page applications
  • Dynamic menu and navigation bars
  • Simple identity and access management with role-based access control
  • Persistent login through the use of token cookies (JWT)
  • Simplified creation of HTML tables from SQL data sources
  • Pagination of table data
  • Logging of debugging output

Hydrogen is designed to support the following high-level non-functional requirements:

  • Usability on both desktop and mobile browsers
  • Ease of setup
  • Security
  • Extensibility via APIs
  • Customization via CSS, JS, PHP
  • Clearly-structured and -commented code
  • Up-to-date library compatibility
